# Antigravity Pulse

A lightweight, real-time quota monitor for Antigravity AI models right in your VS Code status bar.

## Features

- Displays current quotas for up to 6 models (Gemini Pro, Gemini Flash, Claude Sonnet, Claude Opus, GPT-OSS).
- Pin your favorite model to the status bar (with percentage).
- Hover to view the exact reset time for your quotas.
- Click to open a responsive dashboard.

## Requirements

- Antigravity Local Background Engine running on port `31415`.

## Usage

Simply install this extension, and when your local Antigravity server is running, the pulse will start!
